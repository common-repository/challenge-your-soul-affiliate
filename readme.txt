Adds a shortcode and widget to facilitate adding ChallengeYourSoul.com affiliate ads.

== Description ==

A plugin that adds a WordPress shortcode to easily add ChallengeYourSoul affiliate ads in your posts and pages. 
Also includes a widget that displays a random affiliate ad. Based on the affiliate ad script found on the 
[challengeyoursoul](http://multimedia.challengeyoursoul.com/afftools.php) website.

== Installation ==

Please install this plugin in one of the following ways:

FTP

1. Download and extract the ZIP file.
1. Upload the extracted folder to the `/wp-content/plugins/` directory.

Upload via the WordPress dashboard

1. Upload the ZIP file via the upload form at the upload tab in the add plugin page.

In all cases

1. Activate the plugin through the 'Plugins' menu in WordPress
